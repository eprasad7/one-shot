### What was done

- Implemented SWE-agent ACI patterns in deploy/src/runtime/tools.ts:
  - Git tools: git-init, git-status, git-diff, git-commit, git-log, git-branch, git-stash
  - Lint-on-edit: Python (ast.parse), JS/TS (node --check), JSON validation
  - Edit diff output: unified diff shown after every edit
  - Stateful file viewer (view-file): cursor-based, 100-line windows, line numbers
  - read-file: now supports offset/limit pagination with total line count
  - search-file and find-file: new SWE-agent ACI navigation tools
  - Smart grep/glob: counts total matches first, forces query refinement when too many
- Implemented reflection & retry in deploy/src/runtime/edge_graph.ts:
  - FRESH_REFLECT node: confidence scoring (tool failures, warnings, short answers)
  - Reflection retry: if confidence < 0.6, injects guidance and re-routes to LLM (max 1)
  - Tool failure recovery: injects system message guiding agent to try different approach
